# Assets Required

Before building, create these images and place them in this folder:

## Required Files

### icon.png
- Size: 1024 x 1024 pixels
- Format: PNG (no transparency)
- Purpose: App icon shown on home screen

### splash.png
- Size: 1284 x 2778 pixels (or larger)
- Format: PNG
- Purpose: Launch screen while app loads
- Tip: Keep design simple, centered logo works well

### adaptive-icon.png (Android only)
- Size: 1024 x 1024 pixels
- Format: PNG with transparency
- Purpose: Android adaptive icon foreground
- Note: Keep important content in center 66%

### favicon.png (Web only)
- Size: 48 x 48 pixels
- Format: PNG
- Purpose: Browser tab icon
